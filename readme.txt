高铁车站站前广场基站一直存在强干扰，附近用户投诉较多，严重影响用户感知，现场排查测试终端占用3XX-高铁西站LHY-CH-10 小区与3XX-安顺高铁车站站前广场LHY-CH-11小区,同为E频段且存在同覆盖区域，闭塞3XX-高铁西站LHY-CH-10小区后干扰消失，确定为3XX-高铁西站LHY-CH-10。
后台查询两个小区RRU型号不同（高铁西站为RRU3125，高铁车站站前广场为RRU3161），由于RRU3161是双模RRU，为了和TDS空口同步，作了空口提前，而RRU3152为单模RRU，怀疑未作空口提前，RRU3161小区和RRU3152小区空口对不齐,造成干扰。
修改干扰小区的帧偏置,让重叠覆盖小区之间空口上同步,而达到干扰消除,修改命: MODTDDFRAMEOFFSET:TIMEOFFSET=277560;(初始值为0)。
